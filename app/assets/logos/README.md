# happydesigns logo assets

Final logo artwork:

- `happydesigns.svg`: light-surface text-only website logo.
- `happydesigns-dark.svg`: dark-surface text-only website logo.
- `happydesigns-favicon.svg`: favicon and app icon source.
- `happydesigns-favicon.png`: PNG app icon fallback.

Canonical logo colors:

- Graphite: `#242423`
- Warm White: `#FAF7F2`
- Soft Peach Field: `#FFE4D4`
- Coral Dot: `#F28564`

The favicon/app icon is intentionally separate from the website wordmark. Use the text-only logo for website navigation and brand display.

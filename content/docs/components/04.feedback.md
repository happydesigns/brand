Feedback should be calm, specific, and tied to the next useful action. Color should clarify state, not decorate the message.

## Feedback Patterns

::nuxt-ui-docs-link{links="UBanner|https://ui.nuxt.com/docs/components/banner; UAlert|https://ui.nuxt.com/docs/components/alert; UProgress|https://ui.nuxt.com/docs/components/progress; USkeleton|https://ui.nuxt.com/docs/components/skeleton; UToast|https://ui.nuxt.com/docs/components/toast; UEmpty|https://ui.nuxt.com/docs/components/empty"}
::

Use banners for broad notices, alerts for nearby state, progress and skeletons for waiting, toasts for finished actions, and empty states for a clear next step. Each pattern should make the state and the recovery path easier to understand.

Chat and editor components should keep writing, review, and assistive workflows readable. The brand layer should support calm interaction, visible structure, and clear system feedback.

## Chat Workspace

::id-nuxt-ui-docs-link{links="UChatPalette|https://ui.nuxt.com/docs/components/chat-palette; UChatMessages|https://ui.nuxt.com/docs/components/chat-messages"}
::

Chat palettes combine message history, prompt, and submission state. Role, content, and current status should stay easy to scan before adding expressive details.

::component-example{name="chat-palette"}
::

## System Activity

::id-nuxt-ui-docs-link{links="UChatReasoning|https://ui.nuxt.com/docs/components/chat-reasoning; UChatTool|https://ui.nuxt.com/docs/components/chat-tool; UChatShimmer|https://ui.nuxt.com/docs/components/chat-shimmer"}
::

Reasoning and tool displays make system behavior visible without competing with the conversation. Streaming indicators should remain subtle.

happydesigns uses three core logo forms and three standardized lockups. The logo forms are the core identity marks. The lockups are controlled combinations for moments where symbol and name should appear together.

The wordmark and symbol express different sides of the brand. The wordmark is elegant, calm, and typographic. The symbol is warmer, more playful, and design-led. Both represent happydesigns, but they should not be forced together in every context.

## Choosing the right form

Use the wordmark when the name, elegance, and calm recognition matter most.

Use the full symbol when the context is square, compact, avatar-like, or symbolic.

Use the reduced symbol when the full symbol becomes too detailed or technically impractical.

Use a lockup when symbol and wordmark need to be connected.

Use the vertical lockup/signature for spacious brand-owned areas, especially the footer.

## Header usage

Use the wordmark in the main website and documentation header. The header should feel calm, elegant, and professional. The favicon already carries the symbol in browser chrome, so the symbol does not need to be repeated in the default header.

## Footer usage

Use the vertical lockup/signature in footers when space allows. The footer is a brand-owned area and is the right place to connect the expressive symbol with the elegant wordmark. In compact footer layouts, use the wordmark only or the reduced horizontal lockup.

## Small and production contexts

Use the reduced symbol or reduced horizontal lockup when the full symbol would be too detailed, too visually heavy, or impractical to reproduce. Typical examples are pens, stamps, engraving, embroidery, small labels, and limited-color print.

## Color behavior

The full symbol does not change between light and dark mode. It keeps its original white tile, graphite h-shape, soft peach field, and coral dot. It behaves like a branded object rather than a UI surface.

Wordmark, reduced symbol, and lockups have inverse variants for dark surfaces.

Approved logo colors are Graphite `#242423`, Warm White `#FAF7F2`, Soft Peach `#FFE4D4`, and Coral `#F28564`.

## Clearspace

Use the coral dot as the clearspace reference.

Minimum clearspace is `2x` coral dot width.

Preferred clearspace for brand moments is `3-4x` coral dot width.

## Do

- Use the wordmark as the default website/header identity.
- Use the full symbol for favicon, app icon, GitHub/social avatar, and square brand contexts.
- Use the reduced symbol for small, technical, monochrome, or production-constrained contexts.
- Use the vertical lockup/signature in the footer when space allows.
- Use lockups only when the relationship between symbol and name should be made explicit.
- Keep proportions and spacing fixed.

## Do not

- Treat every asset as an equal core identity mark.
- Use the symbol as generic decoration.
- Use the full horizontal lockup in cramped headers.
- Recolor the full symbol for dark mode.
- Use arbitrary palette colors inside the logo.
- Use the reduced symbol as a stylistic replacement for the full symbol in brand moments.
- Repeat multiple logo forms in the same view without a clear reason.

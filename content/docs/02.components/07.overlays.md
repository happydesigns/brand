Overlays should support the current task without making the interface feel heavier than it needs to be.

## Focused Decisions

::id-nuxt-ui-docs-link{links="UModal|https://ui.nuxt.com/docs/components/modal; USlideover|https://ui.nuxt.com/docs/components/slideover; UDrawer|https://ui.nuxt.com/docs/components/drawer"}
::

Modals are for focused decisions, slideovers support secondary work in the current context, and drawers fit compact or mobile-first flows.

::id-nuxt-ui-docs-link{links="UPopover|https://ui.nuxt.com/docs/components/popover; UTooltip|https://ui.nuxt.com/docs/components/tooltip"}

## Overlay Labels

Overlay titles should name the decision or object: `Review pattern`, `Delete token`, or `Project settings`. Avoid generic titles such as `Are you sure?` when the action can be named directly.

Typography defines the reading voice of the brand. The system should feel warm without losing precision.

## Families

### Bricolage Grotesque

Primary sans for headings, navigation, labels, and body copy.

```css
font-family: "Bricolage Grotesque", ui-sans-serif, system-ui, sans-serif;
```

### Geist Mono

Technical mono for code, token names, version labels, and compact metadata.

```css
font-family: "Geist Mono", "SFMono-Regular", ui-monospace, monospace;
```

## Hierarchy

- Page titles should be confident, compact, and direct.
- Body copy should prioritize readability over brand expression.
- Metadata should use the mono stack only when it clarifies structure.
- Keep letter spacing at `0`; do not tighten display text.

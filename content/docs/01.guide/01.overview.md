[`@happydesigns/brand`](https://github.com/happydesigns/brand) is both the happydesigns guide and the installable Nuxt UI layer. Use these pages to make identity decisions, then install the layer when a Nuxt project should carry those decisions in real components.

The guide is built on [`@happydesigns/id`](https://github.com/happydesigns/id), the shared brand-guide foundation. `id` provides reusable guide mechanics; this repository remains the source for happydesigns assets, copy, palette meaning, voice, examples, and final rules.

## Install the layer

::code-group
```bash [pnpm]
pnpm add @happydesigns/brand
```

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['@happydesigns/brand']
})
```
::
